Login & Registration System (PHP)

This is a simple Login and Registration project made using PHP and MySQL.

Features

User can register

User can log in

Password is saved safely

User can log out

Files

register.php – user signup page

login.php – user login page

logout.php – logout page

config.php – database connection

How to Run

Put the project in xampp/htdocs

Start Apache and MySQL

Import the SQL file in phpMyAdmin

Open in browser:

http://localhost/Login & Registration System (PHP)
